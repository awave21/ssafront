#### 2. Исправлен формат возврата данных в sqns_mcp_server.py

**Проблема:** SQNS API возвращает `{'resources': [...]}`, а FastMCP ожидает массив.

**Решение:**
```python
# sqns_list_resources
result = await client.list_resources()
if isinstance(result, dict) and 'resources' in result:
    return result['resources']  # Извлекаем массив
return result if isinstance(result, list) else []
```

#### 3. Исправлено логирование FastMCP

**Проблема:** `AttributeError: 'FastMCP' object has no attribute '_tools'`

**Решение:**
```python
tool_count = len(getattr(mcp, '_tools', [])) or len(getattr(mcp, 'tools', []))
logger.info("sqns_mcp_server_created", tool_count=tool_count)
```
